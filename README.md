# Boothelp

Boothelp is a helper for twitter-bootstrap and font-awesome elements.
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'boothelp'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install boothelp
